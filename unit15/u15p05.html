Step 4: Tracing

In this fourth step you will trace some paths through the circuit. In some cases the timing of this and step 3 can overlap or swap. If you are struggling with any portions of step 3 then it is worth trying this step for that portion of the circuit first to see if it helps you become un-stuck (or perhaps less stuck? The order of these is not important, do whatever is easiest.).

Trace the signal

I typically start by tracing the signal flow through the circuit. The signal refers to the audio and has to get from the input to the output. If you have identified all the sub-circuits in the previous step then this should not be too difficult.

The 1176 preamplifier example shown in figure 5-1 comes relatively simply from the sub-circuits except for the portion of the path that is indicated with the dashed line. This is the feedback path and might not be identified yet. I recommend that you draw (with highlighters, markers, or colored pencils) directly on the paper copy of your schematic for this. It is possible to do with something like Google draw, but experience has shown me that a paper copy works better for groups of students.

Polarity

Also in figure 5-1 you will see images that indicate the signal polarity at every part of the signal path. This is really important with audio circuits. This was the step in the process that I became certain of the "negative feedback" path and this step will often lead to more discoveries. Again, I would draw these directly on the paper schematic.

Trace the DC Current Paths

The signal path goes from the input of each sub-circuit to the output. You should remember from the current tracing that you did in earlier courses that the current path and signal path are different. On a basic common emitter circuit the signal path is base to collector while the DC current path (conventional) is collector to emitter with a tiny bit going base to emitter. If we ignore that tiny bit of current going base to emitter, the current path for each sub-circuit should be separate.

Figure 5-2 shows the DC current paths for the 1176 preamplifier. Note the path gong from the emitter of Q3 to the base of Q2. There was nothing biasing the base of Q2 when we listed the typical components. Tracing the DC path showed us where that DC was coming from to bias the base.

Figure 5-2 also shows the two components in the circuit that have AC flowing through them, C7 and C8.
