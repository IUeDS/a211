a Five Step method

The goal for the rest of the course is to build a method for and practice with analyzing complex audio circuits. We will start by outlining five steps that make up a methodology for circuit analysis. As I outline the process I will be showing images from the preamp section of the 1176 limiter schematic, shown in figure 1-1. When it is closer to the time you will give your presentations I will present an analysis of the 1176 preamp section as a demonstration.

The end goal with this is to define the function of every component in the circuit.

  1. Identify and indicate voltage rails and ground connections on schematic
    • make the schematic easier/quicker to read
    • make it easier to know what connected to ground
    • make it easier to know what is connected to a voltage source
  2. Identify components and their basic properties
    • may seem simple but can be very useful
    • have this written out so it can sit next to the schematic as you work
    • good time to identify any "unknown" components
  3. Identify sub-circuits and their basic properties
    • utilize all that knowledge you have built about how sub-circuits function!
    • this step will often move you forward a lot
  4. Trace signal paths and current paths and indicate signal polarity
  5. Analyze with input signal
    • This will vary a great deal depending on the actual circuit
    • This is where you really start to see how the circuit functions.

Will this be hard?

I have been working with students on this project for many, many years. Typically the hardest part for students is keeping focused on the current steps and not trying to jump ahead. It is easy to get distracted with "what does this do". If you stay focused on the current step of the process the project will go much more easily. This process will allow you to gain a deep understanding of how the circuit functions.
