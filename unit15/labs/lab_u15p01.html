Now Build It!

14-3. On the right you see the schematic from the reading. This has all the features and safety precautions we need for regular use. We need very little of that in the lab, so we are going to build the simplest version of a circuit needed to show how this IC works.

14-4. The function generator in the lab is unbalanced but we want a balanced signal for the preamp input. The simplest way to get a balanced signal is to use a transformer so that is what we will do. Note that this transformer will pass audio, but has limited frequency response and would not be good for much beyond voice. Fine for what we are dong though, and really inexpensive.

14-5. Below, on the left, you see a simplified schematic with the transformer, build this on your breadboard. Note that the transformer is there to balance the function generator, it is not a part of the preamp circuit.
