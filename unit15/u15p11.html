Compressors and Limiters

Let's start with a simple explanation of what compressors and limiters do. This is a very simplified explanation but will work for this course.

Compressors allow you to reduce the dynamic range of a signal. That means that the range from loud to soft will be smaller after the signal goes through the compressor. A limiter is a more extreme version of a compressor.

A block diagram of a basic compressor/limiter is shown in figure 11-1. There are two paths in most compressors (or limiters), they are:

  • Signal Path: This is the actual path of the audio you want to compress. You send a signal into the input and you get that signal out with less dynamic range.
    (***note, this is very much an oversimplification of compression! )
  • Sidechain Path: This is where the audio is converted into something that can be used to control the gain reduction to produce the compressed signal. this portion of the circuit is typically much more complex than the signal path.
  • Control Voltage: The audio in the sidechain path is often converted to DC and used to control the gain of the component that actually changes the gain (see below).

Compression Methods

  • VCA: VCA stands for Voltage Controlled Amplifier and it is an amplifier with a gain that is set by a DC voltage. The audio in the sidechain is converted into DC. The voltage of that DC is related to the volume of the audio. Think of the rectifier circuits we have already looked at. The louder the signal the higher the DC voltage. That DC voltage can be used to control the gain of the VCA.
  • Optical: This type of compressor uses a photo sensitive component. This component changes is properties depending on how much light it is exposed to. A photoresistor for instance will change resistance based on how much light you allow to get to it. This photosensitive component is place in the circuit so that it can control gain (in an OpAmp feedback loop for instance) and then it is exposed to light. The light it is exposed to is created from the audio signal, more audio signal = more light.
  • Vari-mu: This is very similar to a VCA but uses a very special type of tube called a variable mu or remote cutoff. A DC voltage added to the control grid of this tube can control the gain of the tube. As with the VCA (and possible the optical) the audio is converted to a DC voltage and used to control the gain of the vari-mu tubes.
  • FET: This is similar to the optical compressor except that the voltage on the gate of the jFET changes current through the source>drain. This can be used to control gain by affecting feedback or directly controlling the audio signal.
  • Diode Bridge: This is a rarely used type of compression that uses the "limit" function of diodes to control how far the signal is away from a control signal.
