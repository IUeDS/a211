I'll Follow Voltage

One common example of the non-inverting amplifier from page 4 is called a voltage follower. A voltage follower has an output voltage that is the same as the input voltage. the schematic is shown in figure 5-1.

Why would we need an amplifier to have the same voltage at the output and the input?

There are a few main reasons for using a voltage follower:

  1. Isolation: The input and output are isolated from each other. This means that something happening at the output won't affect the input. The input signal may need to be sent to more than one place (a headphone output and an output that goes to speakers, for instance). You don't want the circuits for those two outputs to affect each other, so you could use voltage followers to isolate them from each other.
  2. Current: Though the output voltage matches the input voltage, the output current does not have to match the input current. Although most products are designed for voltage transfer, there are times when current is needed. For those times when you need a circuit to provide more current for the same voltage, a voltage follower is useful
  3. Voltage followers are also the basis for some of the filter circuits we will be investigating.

Gain

Gain is found with the same math, but it is not necessarily as obvious.
$Gain = 1 + \frac {R2}{R1}$

But there is no R1, nor is there an R2!!!!

  • R1: There is nothing connecting the inverting input to ground. With no connection we think of the resistance as being as high as possible, for the math we will say that it is infinite, $R1 = \infty$
  • R2: There is no R2 along the connection from the output to the inverting input, there is just a connection. In that case we say that the resistance is as small as possible, $R2 = 0$

So we are left with $Gain = 1 + \frac {0}{\infty}$
Since $\frac {0}{\infty} = 0$
Gain = 1
